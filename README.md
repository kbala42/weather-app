# Weather App (v1.0)

Basit bir komut satırı hava durumu uygulaması.

## Nasıl Çalışır?

1. Kullanıcıdan şehir adı alır.
2. OpenWeatherMap API'sine istek gönderir.
3. Şehrin anlık sıcaklığını (°C) ekranda gösterir.
4. Hata durumlarında anlaşılır mesajlar verir.

## Çalıştırma

```bash
pip install -r requirements.txt
python weather.py
